# 02. 사용자 시나리오

## 페르소나

- **김순자 (78세, A1)**: 완전틀니 2년차. 스마트폰은 전화·카카오톡 정도만 사용. 시력이 약해 작은 글씨를 못 읽음. 틀니 관리법을 치과에서 들었지만 자꾸 잊어버림.
- **김미선 (52세, A2)**: 순자님의 딸. 직장인이라 매일 전화로 챙기기 어려움. 어머니가 틀니 관리를 잘 하시는지, 치과는 제때 가시는지 걱정.

## 시나리오 1: 최초 설정 (A1 + A2 함께)

1. 미선이 어머니 폰에 앱 설치, 함께 회원가입 (전화번호 기반)
2. "제가 직접 사용해요" 선택 → 온보딩: 이름, 출생연도, 틀니 제작 시기(2024년 3월), 다니는 치과, 알림 시간 확인
3. 온보딩 마지막에 초대코드 6자리 표시 → 미선이 자기 폰에 앱 설치 후 "가족이 도와줘요" 선택, 코드 입력
4. 연결 완료 → 미선 폰에 "어머니와 연결되었습니다" 표시
- **성공 조건**: 10분 내 설정 완료, 이후 순자님 혼자서도 앱 사용 가능

## 시나리오 2: 매일 루틴 (A1 핵심 반복 시나리오)

1. 08:00 알림: "아침 식사 후 틀니를 닦을 시간이에요"
2. 순자님이 알림 탭 → 행동 실행 화면: 큰 글씨로 "주방세제로 1분간 솔질하세요" + 그림
3. [했어요] 버튼 → 완료 애니메이션 + "오늘 2/5 완료"
4. 홈으로 복귀, 다음 할 일(점심 식후 12:30) 표시
- **성공 조건**: 알림→완료까지 탭 3번 이내
- **예외**: 알림을 놓쳐도 홈에서 밀린 할 일 확인 가능. 22시까지 3개 미만 완료 시 미선에게 "오늘 관리가 늦어지고 있어요" 알림

## 시나리오 3: 보호자 일상 확인 (A2)

1. 미선이 퇴근길에 앱 열기 → 홈: "오늘 4/5 완료, 23일 연속" 
2. 주간 리포트: 이번 주 83% 달성, 다음 치과 검진 D-12
3. 안심하고 앱 종료 (1분 이내 확인 완료)
- **성공 조건**: 전화 없이 30초 내 어머니 상태 파악

## 시나리오 4: 응급 상황

1. 순자님 잇몸에 통증 → 홈의 빨간 [도움 요청] → "잇몸이 아파요" 선택
2. 미선 폰에 즉시 알림: "어머니가 잇몸 통증을 알렸어요" + 다니는 치과 전화번호 표시
3. 미선이 치과 예약 → 검진 완료 후 순자님 앱에 검진 기록 → 리콜 주기 재계산
- **성공 조건**: 신고→보호자 인지까지 1분 이내

## 시나리오 5: 치과 검진 리콜

1. 틀니 제작 2년차(안정기) → 6개월 주기 자동 계산
2. 검진 D-7부터 순자님·미선 모두에게 알림
3. 검진 다녀온 후 [검진 받았어요] 기록 → 다음 리콜 자동 갱신
- **성공 조건**: 리콜 놓침 방지 — 검진 초과 시 지속 리마인드
